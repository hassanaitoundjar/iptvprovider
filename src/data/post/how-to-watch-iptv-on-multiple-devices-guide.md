## Network Setup for Best Performance

### Router Placement

**Optimize WiFi coverage:**

- Place router centrally in home
- Elevate router (shelf or wall mount)
- Avoid obstacles (walls, metal, appliances)
- Keep away from interference (microwaves, cordless phones)

### Wired vs WiFi

**Best practice:**

```
Smart TVs / Fire Sticks ‚Üí Wired Ethernet (best)
Mobile devices ‚Üí WiFi 5GHz (good)
Tablets ‚Üí WiFi 5GHz (good)
```

**Priority wired connections:**

1. Living room TV (most used)
2. Master bedroom TV
3. Home theater system

**WiFi is fine for:**

- Smartphones
- Tablets
- Occasional viewing devices

### Router Recommendations

**For 3-5 devices streaming:**

- Dual-band router (2.4GHz + 5GHz)
- WiFi 5 (802.11ac) minimum
- Example: TP-Link Archer A7 ($50-70)

**For 5-10 devices streaming:**

- Tri-band router or Mesh system
- WiFi 6 (802.11ax) recommended
- Example: Google Nest WiFi ($200-300)

**For 10+ devices:**

- Mesh WiFi system (3-pack)
- WiFi 6 with MU-MIMO
- Example: Eero Pro 6 ($350-500)

### Quality of Service (QoS) Settings

**Prioritize IPTV traffic:**

1. Access router admin panel
2. Find QoS settings
3. Set IPTV/streaming as high priority
4. Limit bandwidth for downloads/uploads during prime time

**Example priorities:**

1. Streaming (IPTV) - Highest
2. Video calls - High
3. Web browsing - Medium
4. Downloads - Low
5. Backups - Lowest

## Troubleshooting Multi-Device Issues

### Issue 1: "Maximum Connections Reached"

**Cause:** Too many devices streaming simultaneously

**Solutions:**

1. **Stop a stream:**
   - Close app on one device
   - Or pause playback

2. **Check ghost connections:**
   - Device left on
   - App running in background
   - Force close app on all devices

3. **Upgrade plan:**
   - Need more simultaneous streams?
   - [Upgrade to premium](/pricing)

### Issue 2: Buffering on Some Devices

**Cause:** Network congestion or weak WiFi

**Solutions:**

1. **Prioritize devices:**
   - Use wired connection for main TV
   - Put phones/tablets on WiFi

2. **Reduce quality:**
   - Lower video quality on mobile devices
   - Keep 4K/HD for main TV only

3. **Upgrade internet:**
   - May need faster plan
   - Calculate: devices √ó 10 Mbps

[Complete buffering fix guide](/how-to-fix-iptv-buffering-issues-complete-troubleshooting-guide)

### Issue 3: Credentials Not Working on New Device

**Cause:** Typo in login credentials

**Solutions:**

1. **Double-check credentials:**
   - Copy-paste from email
   - Check for extra spaces
   - Verify server URL is complete

2. **Test on working device:**
   - If works there, credentials are correct
   - Problem is with new device setup

3. **Contact support:**
   - [Get help 24/7](/contact)
   - Verify account status

### Issue 4: Poor Quality on Mobile Devices

**Cause:** Weak WiFi or cellular data

**Solutions:**

1. **Switch to WiFi:**
   - Don't use cellular for streaming
   - Connect to home WiFi

2. **Move closer to router:**
   - Improve signal strength
   - Or use 5GHz WiFi band

3. **Check WiFi speed:**
   - Test with speedtest.net
   - Need 10+ Mbps per device

### Issue 5: Kids Accessing Blocked Content

**Cause:** PIN not set or disabled

**Solutions:**

1. **Enable PIN:**
   - Settings ‚Üí Parental Control
   - Set 4-digit PIN
   - Don't share with kids!

2. **Lock categories:**
   - Adult content: Always locked
   - News/late-night: Optional

3. **Device-level controls:**
   - Kids' tablets: Maximum restrictions
   - Don't give kids access to master account
