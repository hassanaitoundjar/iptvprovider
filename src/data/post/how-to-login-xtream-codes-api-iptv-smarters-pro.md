# How to Login Xtream Codes API on IPTV Smarters Pro

IPTV Smarters Pro supports Xtream Codes API, which is one of the most popular IPTV service protocols. This guide will walk you through the process of logging in with Xtream Codes API credentials to access your IPTV service.

## What is Xtream Codes API?

Xtream Codes API is a protocol used by many IPTV service providers to deliver content to their subscribers. It allows you to access live TV channels, movies, TV series, and catch-up content through a single API endpoint.

## Prerequisites

Before you begin, make sure you have:

- IPTV Smarters Pro installed on your device

  - **Password**

## Step-by-Step Login Process

### Step 1: Launch IPTV Smarters Pro

1. Open the IPTV Smarters Pro app on your device
2. You'll see the main screen with login options

### Step 2: Access Login Options

1. Tap or click on **"Add User"** or **"Login"** button
2. You'll see different login methods available

### Step 3: Select Xtream Codes API

1. Choose **"Login with Xtream Codes API"** or **"Xtream Codes"** option
2. This will open the login form

### Step 4: Enter Your Credentials

Fill in the required fields:

1. **Username**: Enter your IPTV service username
   - Usually provided by your IPTV service provider
   - Case-sensitive, so enter exactly as provided

2. **Password**: Enter your IPTV service password
   - Keep this secure and don't share it
   - Case-sensitive

3. **Server URL**: Enter your IPTV server URL
   - Format: `http://example.com:port` or `https://example.com:port`
   - Include the port number if specified
   - Example: `http://iptvserver.com:8080` or `https://iptvserver.com:25463`

### Step 5: Save and Connect

1. Double-check all credentials for accuracy
2. Click or tap **"Add User"** or **"Login"** button
3. The app will attempt to connect to your IPTV service

### Step 6: Verify Connection

1. Wait for the connection to establish (may take a few seconds)
2. If successful, you'll see your channel list and content
3. If there's an error, check the troubleshooting section below

## Login Format Examples

### Standard Format

```
Username: your_username
Password: your_password
Server URL: http://your-server.com:8080
```

### HTTPS Format

```
Username: your_username
Password: your_password
Server URL: https://your-server.com:25463
```

### With Port Number

```
Username: your_username
Password: your_password
Server URL: http://iptv.example.com:1234
```

## Troubleshooting Login Issues

### "Invalid Credentials" Error

**Possible Causes:**

- Incorrect username or password
- Server URL is wrong
- Account expired or suspended

**Solutions:**

- Double-check all credentials with your IPTV provider
- Verify your subscription is active
- Try copying and pasting credentials to avoid typos

### "Connection Failed" Error

**Possible Causes:**

- Server URL is incorrect
- Internet connection issues
- Server is down or maintenance

**Solutions:**

- Verify the server URL format is correct
- Check your internet connection
- Contact your IPTV provider to confirm server status
- Try using HTTPS instead of HTTP (or vice versa)

### "Timeout" Error

**Possible Causes:**

- Slow internet connection
- Server overloaded
- Firewall blocking connection

**Solutions:**

- Check your internet speed
- Try again later if server is busy
- Check firewall/VPN settings
- Try a different network connection

### App Crashes During Login

**Possible Causes:**

- App version is outdated
- Device compatibility issues
- Corrupted app data

**Solutions:**

- Update IPTV Smarters Pro to the latest version
- Clear app cache and data
- Reinstall the app if necessary
- Check device compatibility
